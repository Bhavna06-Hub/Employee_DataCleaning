# Employee_DataCleaning Project
This repository contains a dataset with messy employee information and a Python script to clean it. The cleaning includes:

- Fixing missing values
- Standardizing date format (Joining_Date)
- Cleaning Salary, City, and Gender columns
- Removing duplicates
- Formatting readable outputs

## Files

- `CleanedMessyData.xlsx` : Cleaned version of the dataset  
- `MessyData.ipynb` : Python code used to clean the data
